About

This is a P.O.C of Weemo Video call integration in eXo iOS application. Weemo SDK needs at least iPhone 4s and iOS 5.1.

How it works

  • You are connected to Weemo cloud using your intranet account some seconds after logging in to intranet.
  • Select Call from Home Menu. Select a contact in Contacts Tab or enter a username in Dial Tab to start a call.
  • The contact list is currently hard-coded. We are waiting for SOC-3848 to display user's connections.

Tips

  • For more details about connection status, check application log by using free Consolekit from App store

Known issues

  • With a slow network, Weemo SDK often disconnects automatically without any notification (yet). When you can't call a user and there is no response, please try to log out and login again.
  • Probably because of connection quality, sometime the call is made but there is only voice and the video view is blank.
  • To receive call, eXo mobile app must be active. TODO: handle incoming call when the app is in background mode.
